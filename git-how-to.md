# Git How-To

## git branch management

list all local branches
```console
git branch
```

create a new branch
```console
git branch <name>
```

checkout a specific branch
```console
git checkout <name>
```

delete a specific branch
```console
git branch -d <name>
```

rename a specific branch
```console
git branch -m <old> <new>
```

shortcut for creating a branch with checkout
```console
git checkout -b <name>
```
